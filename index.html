📝 Cách làm bài tập tương tác

Click vào nút "Làm bài / Xem trước" trên các thẻ tài liệu. Giao diện bài tập HTML sẽ hiển thị trực tiếp trên web giúp bạn trả lời trắc nghiệm, điền khuyết và biết kết quả ngay lập tức.

💾 Cách tải bài tập về máy

Bấm nút "Tải xuống" để lưu bài tập về máy tính hoặc điện thoại. Bạn có thể mở tệp HTML này offline (không cần mạng) bằng bất kỳ trình duyệt nào (Chrome, Safari, Edge) để tự học.

🎨 Infographics & Sơ đồ tư duy

Tải bộ sơ đồ tư duy dạng SVG/PNG sắc nét về làm tài liệu ôn tập nhanh trước kỳ thi. Bạn có thể in màu ra khổ giấy A4 để dán góc học tập.

🚀 Tải tài liệu mới lên Web

Đây là trang web tĩnh chạy bằng GitHub Pages. Bạn chỉ cần sửa tệp data.js để thêm thông tin bài tập, sau đó copy file HTML bài tập mới của bạn vào thư mục data/ tương ứng.
